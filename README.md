# 나의 인생 기록실 — 배포 안내

이 폴더 전체를 Netlify에 올리면, 누구나 회원가입해서 각자의 데이터로 이 앱을 쓸 수 있습니다.

## 1. Netlify에 배포하기
1. https://app.netlify.com 접속 → 로그인(또는 무료 가입)
2. 사이트 목록 화면에서 이 폴더 전체를 그대로 드래그 앤 드롭
   - (더 편한 방법: 이 폴더를 GitHub 저장소에 올린 뒤 "Import from Git"으로 연결하면 이후 자동 배포됩니다)
3. 배포가 끝나면 `xxxx.netlify.app` 같은 주소가 생깁니다.

## 2. 로그인 기능(Identity) 켜기
1. Netlify 사이트 대시보드 → **Site configuration → Identity**
2. **Enable Identity** 클릭
3. Registration 설정을 원하는 대로 선택
   - "Open" — 누구나 이메일로 자유롭게 회원가입
   - "Invite only" — 내가 초대한 사람만 가입 가능
4. 끝입니다. 이제 사이트에 접속하면 로그인/회원가입 화면이 뜹니다.

## 3. 확인하기
- 사이트 주소로 접속 → "회원가입" 버튼으로 계정을 만들고 로그인
- 일정, 투두, 자기계발, 가계부 등 무엇을 입력하든 그 계정에만 저장됩니다
- 다른 사람이 다른 이메일로 회원가입하면, 그 사람 데이터는 완전히 분리되어 저장됩니다 (서로 볼 수 없음)
- 로그인한 채로 다른 기기·다른 브라우저에서 같은 계정으로 로그인하면 데이터가 그대로 이어집니다

## 데이터는 어디에 저장되나요?
- 로그인 인증: **Netlify Identity**
- 실제 일정/투두/가계부 데이터: **Netlify Blobs**(서버 저장소), 사용자 ID별로 완전히 분리 저장
- 헤더의 "⬇ 백업" 버튼으로 언제든 내 데이터를 파일로 내려받아 보관할 수 있습니다

## 참고
- 이 파일(index.html)을 그냥 더블클릭해서 여는 경우 로그인 기능은 작동하지 않습니다. 위 방식대로 **Netlify에 배포해야** 로그인/회원가입이 정상 작동합니다.
- 요금은 무료 플랜으로 충분합니다 (Identity 무제한 사용자, Blobs 기본 제공).
